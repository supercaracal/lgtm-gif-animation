LGTM GIF ANIMATION
======================

Append LGTM frame to gif animation.

## Installation

```
$ git clone https://github.com/supercaracal/lgtm-gif-animation.git
$ cd lgtm-gif-animation
$ make
```

## Usage

```
$ ./lgtm-gif-animation < input.gif > output.gif
```

## Support

`GIF89a` formatted animation file only

## Dependency

None
