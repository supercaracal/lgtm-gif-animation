LGTM GIF Animation
======================

This tool append LGTM frame to animated gif.

## Usage

```
$ git clone https://github.com/supercaracal/lgtm-gif-animation.git
$ cd lgtm-gif-animation
$ make
$ ./lgtm-gif-animation < input.gif > output.gif
```

## Support versions

* `GIF89a`
* `GIF87a`
